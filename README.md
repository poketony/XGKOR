Xenocomp: 아카이브 암호화/복호화 프로그램

Xenolns: 단일 파일 교체용

Xenoiso: iso 자체 리팩 프로그램

Xenolater: 스크립트 번역툴
	
  Delay X: X 프레임(약 30fps 기준) 대기 후 다음 텍스트
	
  New: 같은 창에서 다음 페이지
	
  Close: 대화 종료
	
  Name, PersoX: 플레이어가 정한 이름 / 파티 멤버 이름
	
  Space X: 실제 글자 수보다 창을 더 크게 잡고 싶을 때(가변 값 포함용)
	
  일부 특수 창은 따로 해킹 필요(it_stuff/field)
