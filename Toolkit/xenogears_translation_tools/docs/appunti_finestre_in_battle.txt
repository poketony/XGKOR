Le informazioni sono contenuti nei file pari dell'archivio 3088.
Ogni file pari è collegato al successivo dispari (dunque lo 0 è
collegato all'1, è così via).
Nei primi byte (circa 160$?) si trovano le informazioni quali
avatar ed altro.
